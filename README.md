# NI VeriStand Custom Device Wizard

## Creating a Custom Device

This repository provides two project templates for creating custom devices:

- **Classic Custom Device** — Use this template for standard custom devices based on the classic template. See the [Custom Device Classic README](Custom%20Device%20Classic/README.md) for details.
- **Express Custom Device** — Use this template for custom devices based on the express template. See the [Custom Device Express README](Custom%20Device%20Express/README.md) for details.
